# Who for Shadowsocks-libev

## What can I do?

Show who login with Shadowsocks-libev.

port and counter

## Installation

Clone to anywhere:

``` shell
git clone https://github.com/ITXiaoPang/who_ss_libev.git
cd who_ss_libev
```

Add exec permission

`chmod +x wss`

Copy to path:

`cp -a wss /usr/bin `
